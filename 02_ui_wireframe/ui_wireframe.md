# 02. 화면 설계서 (UI Wireframe)

## 기준 정보

- 기준 브랜치: `develop`
- 기준 템플릿: `skn22_4th_prj/templates/index.html`, `symptom_result.html`, `symptom_products_page.html`

## 1) 화면 목록

- 메인 페이지: `/`
- 증상/제품 검색 결과: `/smart-search/?q=...`
- 제품 추천 상세: `/smart-search-products/`

## 2) 사용자 흐름

```text
[메인]
  ├─ 증상/제품 검색 → [결과 페이지]
  │                    ├─ 성분/주의 안내
  │                    └─ 상담 메모(복사)
  └─ 내 위치 찾기 → 근처 약국 찾기 → [지도 + 목록]
                            └─ 지도 이동 시 재조회
```

## 3) 메인 페이지 와이어프레임

```text
+--------------------------------------------------------------+
| Header(브랜드/로그인/회원가입/프로필)                        |
+--------------------------------------------------------------+
| Hero(검색창 + 추천 검색어 + 소개 카드)                        |
+--------------------------------------------------------------+
| Feature Cards                                                 |
+--------------------------------------------------------------+
| [내 위치 찾기] [근처 약국 찾기]                               |
| 상태 메시지 / 현재 좌표                                       |
| 지도(상단): 현재 위치(빨간핀) + 약국핀                         |
| 목록(하단): 약국명/주소/전화/운영시간/영업상태                 |
+--------------------------------------------------------------+
```

## 4) 결과 페이지 와이어프레임

```text
+--------------------------------------------------------------+
| 뒤로가기 / 증상 분석 결과 / 제품추천 버튼                      |
+--------------------------------------------------------------+
| 의사/약사 상담용 영어 메모 + 메모 복사                         |
+--------------------------------------------------------------+
| 복용 가능 성분 카드 + DUR 안내                                |
+--------------------------------------------------------------+
```

## 5) 인터랙션 정의

- 약국 핀 hover: 툴팁 표시
- 약국 핀 click: 팝업 상세 표시
- 지도 moveend: 현재 영역 기준 약국 재조회
- 오류 상태: 권한 거부/시간초과/HTTPS 제한 문구 분리
